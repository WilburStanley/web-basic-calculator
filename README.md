# web-basic-calculator


---

## 🚀 Getting Started

1. **Clone or Download** this repository.

2. **Open `calculator.html`** in any modern web browser.

   No build tools or servers needed—it's 100% client-side.

---

## 📸 UI Overview

- Clean and modern interface
- Large input display
- Clearly labeled buttons for intuitive use
- Smooth animations on hover and click

---

## 🛠️ How It Works

- The user inputs numbers and operations using on-screen buttons.
- The input is displayed in a read-only field.
- JavaScript handles event listeners and performs calculations.
- Results are displayed instantly upon pressing `=`, with support for chaining operations.

---

## 📌 Notes

- Button styles are customizable via CSS variables in `:root`.
- Responsive layout uses `flex` and `grid` for adaptability.
- All logic is handled in `calculator.js` without external libraries.
